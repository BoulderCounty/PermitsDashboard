About

A collection of visualization examples leveraging the data and APIs of CivicData.com and Boulder County, CO Building Permit BLDS data.

These examples were built with:

  • Bootstrap: Makes building great looking websites easy.
  • jQuery: Still the awesomest JavaScript framework.
  • D3.js: JavaScript library for manipulating documents based on data.
  • C3.js: Chart library for D3.
  • CartoDB: Easily create amazing maps.
  • Start Bootstrap - SB Admin 2 template: An incredible dashboard template.
  • Moment.js: Parse, validate, manipulate, and display dates in JavaScript.

Built by: Seth Axthelm.
