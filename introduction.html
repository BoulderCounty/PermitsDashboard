Introductory Tour

Navigation

The top row of the dashboard - high level data story (From left to right)
  • The Navigation Pane
  • Time-scale menu: how much data to show
  • Info-boxes: summary information
The pie chart: "Permits by Type"
  • Shows the breakdown of permit Types
  • Selecting a 'slice' of the pie reloads the bar chart with the single, selected Type
  • Type selection triggers the appearance of the Sub-Type menu
    1. the title of the bar chart collapses to reveal a toggle button labeled ‘Subtype(s)’
    2. the graph then alternates between Type totals and a Subtype breakdown
The bar chart: "Applications..."
  • Scrolling over the different bars presents some summary info
  • The button changes from 'Subtype(s)' to 'Total', indicating what will be shown
      o total of that Type per time period (the initial state)
      o a breakdown of the Type into Sub-Types
  • When 'SubType(s)' are visible – when the key details the Sub-Type names - a drop down menu appears next to the 'Totals' button
  • This menu will reload the bar chart with the selected Sub-Type - a complete drill down!
Dynamic "Recent Permit Applications" list
  • Links to the project documentation - allows quick access to recently filed applications
  • The list is static and does not interact with the user - only listing the last ten projects
  • However, with future implementations of this website, the list will change to reflect the data shown in the bar chart; i.e. by drilling down in the bar chart, the ten most recent filings for only the selected Sub-Type will be shown. In addition, the ideal situation will be to make the 'bars' in the bar chart interactive with this list. Selecting a bar will present a scrollable list of all Types or Sub-Types selected within that specific time-frame.

Now dig in!!

Highlighting some examples showing off the data
  • A data landscape: A snapshot of the information at your fingertips.

    • Select the 'Building' (blue) slice of the pie, and you can see an annual cycle in the data.

  • Shorter time frame: While data is normally broken down by month, for time periods less than six months, it is broken down by week

    • Again, select the 'Building' slice, and notice that an obvious pattern in absent.

    Now, click 'Subtype(s)' and then select 'Accessory Agricultural Building' from the drop-down menu. Notice the spike in the daily data.

  • The 2103 Flood

    • Select the 'Mechanical' slice (green)

    Look at the 'Furnaces' sub-type. The peak in October of 2013 is due to the flood.

  • Daily data: Daily data shown in one month.

    • You might also be interested in the patterns show by the 'Air Conditioning' subtype. See if you can find it without directions.
